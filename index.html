Linguaggi 🎓

Queste sono le tecnologie che ho usato per creare queste applicazioni :

Python

Python è un linguaggio di programmazione di alto livello rispetto alla maggior parte degli altri linguaggi, orientato a oggetti, utilizzato nel campo dello sviluppo web e blockchain. Le caratteristiche più immediatamente riconoscibili di Python sono le variabili non tipizzate e l'uso dell'indentazione per la sintassi delle specifiche, al posto delle più comuni parentesi.

Django

Django è un web framework con licenza open source per lo sviluppo di applicazioni web, scritto in linguaggio Python, seguendo il paradigma "Model-Template-View". L'orientamento che ha Django nella gestione dei siti di notizie è evidente dal suo design, poiché fornisce un certo numero di funzionalità che facilitano lo sviluppo rapido di applicazioni per la gestione di contenuti Web.

Solidity (attualmente lo sto studiando)

Solidity è un linguaggio di programmazione orientato agli oggetti per la scrittura di contratti intelligenti.Viene utilizzato per implementare contratti intelligenti su varie piattaforme blockchain, in particolare Ethereum. Sono state supportate variabili membro complesse per i contratti,inclusi mapping e strutture arbitrariamente gerarchiche.
